# Opis projektu
projekt wyświetla produkty oraz ich cene.

# Instrukcja uruchomienia
1.otworzyć folder w vs code
2.otworzyć nowy terminal i włączyć backend poniższymi komendami
cd lab5/sample-ci/backend/ProductApi
dotnet restore
dotnet run
3.otworzyć kolejny terminal i włączyć frontend poniższymi komendami
cd lab5/sample-ci/frontend
npx serve .
4.wkleić w przeglądarke: http://localhost:3000 żeby otworzyć frontend, na http://localhost:5000 otwiera sie backend
